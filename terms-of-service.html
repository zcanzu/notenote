4. User Accounts and Matching

4.1 Account Creation

When you first use the App, a unique user account is automatically created. You are assigned a unique 6-character code that expires after 30 seconds. This code is used to match with another user.

4.2 Matching Process

To use the App, you must match with another user by:

  • Sharing your unique code with another user, or
  • Scanning another user's QR code, or
  • Entering another user's code manually

Once matched, you and your partner can share notes and drawings. Each match (channel) has an expiration time that you can set (1-12 hours). After expiration, the match automatically ends.

4.3 Matching Restrictions
